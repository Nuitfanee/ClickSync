# Click Sync
A console application based on the Web HID API for configuring and managing multiple brands of mice.

## 功能特性

### 🎮 核心功能

- **按键映射**：自定义鼠标按键功能，支持多种按键映射选项
- **DPI 设置**：配置多个 DPI 档位（1-6档），自定义 DPI 范围和档位值
- **基础性能**：调整回报率（125Hz - 8000Hz）和性能模式（低功耗/标准/竞技/超频）
- **高级参数**：
  - 电源按键参数（休眠时间、防抖）
  - 传感器参数（角度修正、手感值）
  - 高级性能开关（移动同步、直线修正、纹波修正、辅助压枪等）
  - LED 指示灯控制
  - LOD 静默高度设置

### 🧪 测试工具

- **双击检测**：检测鼠标按键的双击问题，统计按下/抬起次数和时间间隔
- **轮询率检测**：实时检测鼠标回报率，显示峰值、平均值、抖动等指标
- **灵敏度匹配**：帮助在不同鼠标之间匹配相同的移动灵敏度
- **鼠标角度校准**：自动检测并校准鼠标的握持角度

## 支持的设备

- **Chaos**：VID `0x1915`
- **Rapoo（雷柏）**：VID `0x24ae`
- **ATK**：VID `0x373b`

## 技术栈

- **纯前端应用**：HTML5 + CSS3 + JavaScript (ES6+)
- **Web HID API**：用于与鼠标设备通信
- **无依赖**：无需 Node.js 或构建工具，直接在浏览器中运行

## 使用方法

### 1. 打开应用

直接在浏览器中打开 `index.html` 文件，或通过 Web 服务器访问。

### 2. 连接设备

1. 点击起始页的触发区域
2. 浏览器会弹出设备选择对话框
3. 选择你的鼠标设备并连接

### 3. 配置设备

连接成功后，可以通过左侧导航菜单访问各个功能模块：

- **按键设置**：点击鼠标图片上的按键点，选择要映射的功能
- **DPI设置**：配置 DPI 档位数量和范围，设置每个档位的具体值
- **基础性能**：选择性能模式和回报率
- **高级参数**：调整传感器和高级功能参数
- **测试工具**：使用各种测试工具检测和校准鼠标

## 浏览器要求

- **Chrome/Edge 89+**：完整支持 Web HID API
- **其他浏览器**：可能不支持 Web HID API，无法使用

### 注意事项

- 使用轮询率检测时，建议关闭浏览器的鼠标手势功能，只连接一个鼠标。
- 某些功能需要关闭 Windows 的"增强指针精确度"选项
- 测试工具建议在 Chrome/Edge 浏览器中使用以获得最佳体验

## 开发说明

### 设备适配器架构

项目采用适配器模式，支持多种设备类型：

- 每个设备品牌有独立的协议实现文件（`protocol_api_*.js`）
- 通过 `DeviceAdapters` 注册表统一管理设备适配器
- 设备自动识别，根据 VID 和 Usage Page 匹配对应协议

### UI 变体系统

- 不同设备可以拥有独立的 UI 样式和配置
- 通过 `DeviceUI.applyVariant()` 应用设备特定的 UI 变体
- 支持设备家族（如 Rapoo 家族）的兼容性处理

## 注意事项

⚠️ **重要提示**

- 本应用需要浏览器支持 Web HID API，目前仅 Chrome/Edge 89+ 支持
- 修改设备参数前请确保了解各项设置的含义
- 某些高级功能（如辅助压枪）可能在某些游戏中被视为违规，请谨慎使用
- 建议在使用前备份设备的原始配置
